# SkyAscent
Game Air Shooter

Bersiaplah untuk menghadapi Sky Ascent, game VR tembak-menembak yang mendebarkan! 
Tembak meteor jatuh, dapatkan skor tertinggi, dan rasakan sensasi imersif dalam petualangan langit yang tak terlupakan. 
Segera mainkan dan tunjukkan kemampuanmu!

Cara Bermain:
1. Persiapkan Perangkat VR
2. Mulai Permainan
3. Posisikan dan Arahkan Pistol Virtual
4. Tembak Meteor
5. Peroleh Skor Tertinggi

Berikut adalah petunjuk untuk mengunduh dan menginstal Sky Ascent VR APK (SideQuest):
1. Unduh APK Sky Ascent VR
2. Izinkan Sumber Tidak Dikenal
3. Temukan File APK
4. Instal APK
5. Selesai
